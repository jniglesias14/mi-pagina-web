Mobile Threat Analysis Platform

🔹 Descripción


Aplicación móvil desarrollada en colaboración con la Universidad de Informática de Valladolid para el análisis y detección de amenazas en aplicaciones Android. La plataforma permite visualizar de forma intuitiva información de seguridad obtenida a partir de motores de análisis especializados, facilitando la identificación de riesgos y comportamientos potencialmente maliciosos.


La aplicación ofrece un sistema flexible de visualización donde cada usuario puede personalizar cómo desea consultar los resultados del análisis, generando dinámicamente dashboards, gráficos y métricas adaptadas a sus necesidades.


Además de consultar aplicaciones previamente analizadas, la plataforma permite cargar archivos APK para iniciar nuevos procesos de análisis, ampliando continuamente la base de conocimiento y proporcionando información detallada sobre permisos, comportamientos, indicadores de riesgo y categorías de seguridad.
